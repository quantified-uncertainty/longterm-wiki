## Parallelization Guide

**Can run simultaneously right now:**
- #1166 (backfill relatedEntities) — independent
- #1168 (cleanup/deletion) — independent
- #1169 (extraction prompt fixes) — independent

**Must be sequential:**
- #1166 + #1168 → then #1167 (verification needs clean data)
- #1167 → then #1170 (improve pipeline needs sourced claims)
