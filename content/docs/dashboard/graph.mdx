This page previously hosted an interactive entity relationship graph visualization. That feature relied on legacy components that are no longer available.

Entity and relationship data is now accessible through the internal dashboards:

- **Entities Dashboard** -- browse all entities, types, and metadata
- **Page Coverage Dashboard** -- structural completeness and link health across pages

To explore entity relationships programmatically, use:

```bash
pnpm crux query search "topic"
```
